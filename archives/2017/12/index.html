package.json 字段详解

每个项目的根目录下，一般都有一个package.json文件，定义了这个项目所需要的各种模块，以及项目的配置信息（比如名称，版本，许可证等元数据）。npm install命令根据这个配置文件，自动下载所需的模块，也就是配置项目所需的运行和开发环境。 name 都是小写， 只能是一个词，不能有空格 _和-都是允许的 version name 包的名字，命名规则：都是小写，数字，-,_, 禁止
